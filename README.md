# Financial Report Analysis Skill for MiniMax

This repository uses the MiniMax-compatible skill layout:

```text
skills/financial-report-analysis-skill/
├── SKILL.md
├── references/
└── scripts/
    ├── financial_report_skill/
    └── requirements.txt
```

Import URL recommended for MiniMax:

```text
https://github.com/Sakura-Yae/financial-report-analysis-skill/tree/main/skills/financial-report-analysis-skill
```

If importing the whole repository works, this URL is also acceptable:

```text
https://github.com/Sakura-Yae/financial-report-analysis-skill
```

The agent runtime must support Python execution, file read/write access, and the packages `pandas`, `numpy`, and `openpyxl`.
