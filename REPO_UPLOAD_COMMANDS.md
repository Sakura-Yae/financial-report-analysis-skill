# GitHub 上传命令

在解压后的本目录中执行：

```bash
git init
git add .
git commit -m "Initial financial report analysis skill"
git branch -M main
git remote add origin https://github.com/<your-account>/financial-report-analysis-skill.git
git push -u origin main
```

注意：

- 仓库根目录应直接包含 `SKILL.md`。
- 不要上传真实财务报表或运行输出。
- 如果仓库是 Private，请确认 MiniMax 能访问该仓库。
